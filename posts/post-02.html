Requirement

  • Godaddy Delux Shared Hosting
  • SSH Access to the account
  • Ready to Host Node Web App

I have sample node web server app that you can download at github: Node Web Server App

Process

So now let go straight of how to set up.

Go to Godaddy hosting cPanel and enable SSH for your hosting account. Therefore, you suppose to have now: 

  • Public IP address of your hosting server
  • Username of SSH
  • Password of SSH - change it if you don't know
  • SSH Port Number

 

SSH to the Hosting Server:

  1. `ssh -p port username@IpAddress` Then provide the password when asked
  2. Installing the NVM on your Server Run this first
    `wget -qO- https://cdn.rawgit.com/creationix/nvm/master/install.sh | bash`
    then run this `nvm install stable`
  3. Check if `node` and `npm` are installed: Please exit from ssh and login again before change take effect. `node -v` `npm -v`
  4. Changing the `.htaccess` file:
    1. `cd ~/public_html`
    2. `nano .htaccess`
    3. then copy the below code into the file: ```
      DirectoryIndex disabled
      RewriteEngine On
      RewriteRule ^$ http://127.0.0.1:XXXXX/ [P,L]
      RewriteCond %{REQUEST_FILENAME} !-f RewriteCond %{REQUEST_FILENAME} !-d RewriteRule ^(.*)$ http://127.0.0.1:XXXXX/$1 [P,L] 
    4. Save file and then go back to terminal
    5. Clone sample Node App from Node Web Server App by
      `git clone https://github.com/loynin/node-web-server.git nodes` wait until it finish.
    6. Install the dependencies:
      `cd nodes`
      `npm install`
      wait until it finish
    7. Run the Node App
      From `nodes` directory run `node server.js`
    8. Keep Node Running forever after terminal is closed:
      `node app.js &`
      or install `npm install -g forever` then start the app by `forever start server.js`
    9. if you want to stop the app: `forever stop server.js`
    10. Have fun for your new app
